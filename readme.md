Функциональная задача:

Получение посылки в почтамате происходит по коду, приходящему в смс. При получении посылки, пользователь вводит код и затем происходит списание средств с баналса пользователя, выдается посылка и пользователь видит сообщение вида “Заберите посылку {id посылки} из ячейки №{номер ячейки}. Ваш счет составляет: {баланс}ед.”

Решите задачу со следующими требованиями:


пользователь имеет сумму 500ед на счете
стоимость посылки 100ед
смс: посылка находится в ячейке код для получения A001DFX0

почтамат имеет вид - [null, null, null, “1432HGF” , null]

В условии проверьте, если  код  получения этой посылки в базе данных равен коду из смс, то выдаем посылку - списываем средства и выводим в консоль данные о нашей посылке, в формате:
“Заберите посылку 1432HGF из ячейки №4. Ваш счет составляет: 400ед.”

* Рекомендации:

- тестовые данные сохраняйте в  переменные

- учтите, что данные, которые вводит пользователь, должны совпасть с данными в базе данных

- в базе данных можно найти код, номер ячейки, стоимость по id посылки

- значения "из базы данных"  храните в переменных


Подсказка - шаги реализации (кликнуть, чтобы раскрыть)
1. Сначала задайте в переменные данные пользователя, стоимость посылки, смс с кодом, код в базе дынных и состояние почтамата.
2. Затем проведите проверку: если код посылки в базе данных, соответствует введенному коду из смс, то выдайте посылку, списав средства с баланса пользователя.
3. В конце выведите соответствующее сообщение в консоль.
